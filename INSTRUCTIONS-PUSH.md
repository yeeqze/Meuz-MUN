### 3️⃣ Ouvrez GitHub Desktop

### 4️⃣ Vous devriez voir BEAUCOUP de changements
GitHub Desktop va détecter tous les fichiers modifiés :
- ✏️ Modified: index.html, about.html, style.css, script.js...
- ➕ Added: (si nouveaux fichiers)

### 5️⃣ Commit les changements
**Dans GitHub Desktop :**
- En bas à gauche, dans "Summary", écrivez :
  ```

  ```
- Dans "Description" (optionnel) :
  ```
  - Gradient animé + particules interactives
  - Glassmorphism sur toutes les pages
  - Cards flip 3D
  - Timeline interactive
  - Curseur custom
  - Dark mode complet
  - Animations au scroll
  ```

### 6️⃣ Cliquez sur "Commit to main"

### 7️⃣ Push origin
- Cliquez sur le gros bouton bleu "Push origin" en haut
